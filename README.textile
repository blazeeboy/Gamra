h1. Gamra

h2. Description

Gamra is a graphics application aim to replace fireworks for bothlinux, windows, the application is a targeting webdesigners 

h2. Tools and libraries

* Python
* wxpython
* cairo

h2. Requirements

* vector illustration
* Raster images modefication
* ease of use

h2. Developers

Faculty of computer and informatics studentsSuez canal university, Egypt the project is developed as summer training course 2009/2010

h2. How to contribute 

you can fork that repository by clicking the fork button above and head to : 
http://github.com/blazeeboy/Gamra/issues you'll find tasks explained, don't worry it's very simple  for wxpython and cairo developers. you can choose a task and comment that you'll do it, when you done send pull request to me (blazeeboy) and i'll review your commit and merge it if valid.
